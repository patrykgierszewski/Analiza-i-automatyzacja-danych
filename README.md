Analizuje oraz przerabiam różne data sety, ściągam brudne dane, zajmuje się pełnym sprzątaniem, dodawaniem nowych mierników. Wszystko jest dokonywane na podstawie komend, power query, pivot tables oraz wykończone dashboardami z interaktywnymi slicerami.
Projekt 1 
Cel: Posprzątanie brudnych danych, dodanie przydatnych mierników logistycznych (koszty amortyzacyjne, koszty paliwa oraz koszty pracownika + wyliczenie marży). Zakończone power pivot table który pokazuje nieefektywne trasy które wymagają korekty osmr time oraz actual time.
